# todo-cli-python

A simple command-line based To-Do List application built using Python.
This project allows users to manage daily tasks efficiently with file storage support.

---

## Features

*  Add new tasks
*  View all tasks
*  Delete tasks
*  Persistent storage using file handling (`tasks.txt`)
*  Input validation for better user experience

---

## Technologies Used

* Python
* File Handling
* Lists, Loops, Functions
* Exception Handling
